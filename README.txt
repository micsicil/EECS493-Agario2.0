Instructions for initial setup (only has to be done once):
1. Setup git repo "EECS493-Agario2.0"
2. Install Meteor: https://www.meteor.com/install
3. Navigate to "EECS493-Agario2.0/app/"
4. Run the following command:
    $ meteor npm install three nodejs-physijs
5. Navigate to "EECS493-Agario2.0/app/node_modules/nodejs-physijs/nodejs/libs/"
6. Replace the "physi.js" file installed there with the one at the root of the git directory

Instructions for running the app locally after setup:
1. Navigate to "EECS493-Agario2.0/app/"
2. Run the following command:
    $ meteor
3. Navigate to http://localhost:3000/

Access deployed version at:
https://agar2-0.herokuapp.com
